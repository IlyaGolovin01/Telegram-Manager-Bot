# Telegram Chat Manager Bot (НКВД)

Telegram-бот для управления чатами в стиле СССР. Вместо стандартных формулировок используются термины НКВД, выговоры, аресты и расстрелы.

## Функции

### Для всех граждан
- `/start` — приветствие
- `/help` — список команд
- `/profile [@user]` — личное дело

### Для надзирателей и выше
- `/mute @user 10m причина` — заткнуть товарища
- `/unmute @user` — разрешить говорить
- `/warn @user причина` — вынести выговор
- `/unwarn @user` — снять выговор
- `/warnings @user` — список выговоров

### Для комиссаров и выше
- `/ban @user причина` — арестовать / расстрелять товарища
- `/unban @user` — реабилитировать
- `/kick @user причина` — депортировать
- `/clear N` — конфисковать N сообщений
- `/admins` — партийная номенклатура
- `/antiadvertising` — настройка антирекламы

### Для владельца
- `/addadmin @user` — назначить комиссаром
- `/removeadmin @user` — разжаловать в граждане
- `/setrole @user роль` — изменить роль
- `/broadcast текст` — массовая рассылка
- `/stats` — статистика НКВД

## Антиреклама

Фильтрация антисоветской пропаганды (ссылки и репосты) для граждан ниже ранга `admin`.

Настройка через `/antiadvertising`:
- `/antiadvertising on/off` — включить/выключить
- `/antiadvertising links on/off` — блокировка ссылок
- `/antiadvertising forwards on/off` — блокировка репостов
- `/antiadvertising action delete|warn|ban` — действие при нарушении

## Роли

| Роль        | Уровень |
|-------------|---------|
| Гражданин   | 1 |
| Надзиратель | 2 |
| Комиссар    | 3 |
| Владелец    | 4 |

Назначение ролей в чатах происходит автоматически: создатель чата получает `owner`, администраторы — `admin`.

## Установка

1. Клонировать репозиторий
2. Установить зависимости:  
   `pip install -r requirements.txt`
3. Создать `config.py`:
   - `BOT_TOKEN` — токен от @BotFather
   - `OWNER_ID` — Telegram ID владельца
4. Запустить: `python main.py`

## Зависимости

- Python 3.8+
- pyTelegramBotAPI
