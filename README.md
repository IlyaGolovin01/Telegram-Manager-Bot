# Telegram Chat Manager Bot

Telegram-бот для управления чатами с системой ролей, модерацией и антирекламой.

## Функции

### Для всех пользователей
- `/start` — приветствие
- `/help` — список команд
- `/profile [@user]` — профиль пользователя

### Для модераторов и выше
- `/mute @user 10m причина` — заглушить пользователя
- `/unmute @user` — снять заглушение
- `/warn @user причина` — выдать предупреждение
- `/unwarn @user` — снять предупреждение
- `/warnings @user` — список предупреждений

### Для администраторов и выше
- `/ban @user причина` — заблокировать пользователя
- `/unban @user` — разблокировать
- `/kick @user причина` — кикнуть
- `/clear N` — удалить N сообщений
- `/admins` — список администрации
- `/antiadvertising` — настройка антирекламы

### Для владельца
- `/addadmin @user` — назначить администратора
- `/removeadmin @user` — снять администратора
- `/setrole @user роль` — изменить роль
- `/broadcast текст` — массовая рассылка
- `/stats` — статистика бота

## Антиреклама

Фильтрация сообщений с ссылками и репостами для пользователей ниже ранга `admin`.

Настройка через `/antiadvertising`:
- `/antiadvertising on/off` — включить/выключить
- `/antiadvertising links on/off` — блокировка ссылок
- `/antiadvertising forwards on/off` — блокировка репостов
- `/antiadvertising action delete|warn|ban` — действие при нарушении

## Роли

| Роль       | Уровень |
|------------|---------|
| Пользователь | 1 |
| Модератор    | 2 |
| Администратор| 3 |
| Владелец     | 4 |

Назначение ролей в чатах происходит автоматически: создатель чата получает `owner`, администраторы — `admin`.

## Установка

1. Клонировать репозиторий
2. Установить зависимости:  
   `pip install -r requirements.txt`
3. Настроить `config.py`:
   - `BOT_TOKEN` — токен от @BotFather
   - `OWNER_ID` — Telegram ID владельца
4. Запустить: `python main.py`

## Зависимости

- Python 3.8+
- pyTelegramBotAPI
